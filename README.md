# test_Finbridge

## Развертывание проекта
```
В корне проекта выполнить:
composer install
npm install

Настроить подключчение к БД в файле
/config/db.php

Настроить адрес сервера в файле
/app/app.js (переменная baseURL)

Выполнить миграции:
php yii migrate

Собрать
yarn mix

Настроить web-сервер, чтобы его корень указывал на папку /web

Для задания создан Telegram бот
@TestFinbridgeBot

Сообщения отправляются на мой адрес @valentin.ivlev
Чтобы изменить получателя, необходимо
подписаться на бот (найти его и отправить сообщение /start),
затем в контроллере (/controllers/MessageController.php)
изменить идентификатор чата получателя
(параметр TELEGRAM_CHATID в методе actionSend).
Узнать свой chatid можно воспользовавшись ботом @ShowJsonBot.
Напишите любое сообщение этому боту. В ответ вам придут некие
данные в формате JSON. Найдите поле chat, а в нём поле id.
```
